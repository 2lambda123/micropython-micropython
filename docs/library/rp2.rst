:mod:`rp2` --- functionality specific to the RP2040
===================================================

.. module:: rp2
    :synopsis: functionality specific to the RP2

The ``rp2`` module contains functions and classes specific to the RP2040, as used in the Raspberry Pi Pico.

See the `RP2040 Python datasheet <https://datasheets.raspberrypi.org/pico/raspberry-pi-pico-python-sdk.pdf>`_ for more information, and `pico-micropython-examples <https://github.com/raspberrypi/pico-micropython-examples/tree/master/pio>`_ for example code.
